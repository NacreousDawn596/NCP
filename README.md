# NCP
NCP or nac's chat project is a little chat app made with python by him, where everyone can talk :D

# HOW TO INSTALL IT

```sh
sudo apt-get install git
git clone https://github.com/NacreousDawn596/NCP
cd NCP
chmod u+x setup.sh
./setup.sh
```

# HOW TO USE IT

```sh
python3 main.py
```
then enjoy! :)
